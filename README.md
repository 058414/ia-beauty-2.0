## ✨ O Que É IA BEAUTY 2.0?

Uma skill de análise facial que ajuda mulheres a escolher o corte de cabelo ideal baseado em assimetria facial, compensação visual, SER vs ESTAR, gola e adereços.

**Resultado Final:** Protocolo PDF personalizado com recomendação de corte

---

## 🎯 O Fluxo Completo (User Journey)

### **Q1: ROSTO** ✅ Completo
- Capturar foto + análise de assimetria (MediaPipe)
- Gerar 3 meias-faces
- Reflexão SER vs ESTAR (robusta com 3 versões)

### **Q2: GOLA + ADEREÇOS** ✅ Completo
- Explorar tipos de gola (U, V, Quadrada)
- Explorar adereços (Brincos, Óculos, Colares)
- **Conceito de VOLUME DE INFORMAÇÃO** = Formato + Densidade
- Bridge de Sobrancelhas + as 3 regiões do rosto

### **Q3: CORPO** 🔄 Em Desenvolvimento
- Análise de biotipo + linhas do corpo

### **Q4: SÍNTESE FINAL** ⏳ Próximo
- Consolidar + Gerar PDF
